#  Week 7 Assignment – K-Means Clustering

##  Overview
This assignment focuses on **K-Means Clustering**, an unsupervised machine learning technique used to group similar data points into clusters.  
The objective is to understand how clustering works and how it can be applied to identify patterns in unlabeled data.

---

##  Objectives
- Understand the concept of **unsupervised learning**
- Implement **K-Means clustering**
- Determine the optimal number of clusters
- Visualize and interpret clustering results

---

- Python  
- NumPy  
- Pandas  
- Matplotlib / Seaborn
